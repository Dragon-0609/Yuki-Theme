How to install Как установить About project О проекте
Installation:
Plugin Installation Program Installation
Plugin Installation:

1. Extract from yuki_theme_plugin.zip to pascal directory

2. On open, the yuki theme's icon will be shown in the center. After, click to the icon on top menu

3. A new window will be shown

4. Select theme

5. Click select

6. The theme will be applied

6.1. If the theme is applied, but editor is still white, restart pascal

7. Enjoy the theme

Program Installation:

1. Extract yuki_theme.zip anywhere

2. Open Yuki Theme.exe. Select theme. Click export

3. You will be asked to set path to the pascal directory. After that a message will be shown

4. Open or restart pascal

Установка:
Установка Плагина Установка Программы
Установка Плагина:

1. Распокуйте из yuki_theme_plugin.zip в папку pascal

2. При открытии, будет показана иконка yuki theme в центре. После, нажмите на иконку в верхном меню

3. Покажется новое окно

4. Выберите тему

5. Нажмите "select"

6. Тема будет применена

6.1. Если тема применена, но редактор всё ещё белый, тогда перезапустите pascal

7. Наслаждайтесь темой

Установка Программы:

1. Распокуйте yuki_theme.zip в любом месте

2. Откройте Yuki Theme.exe. Выберите тему. Нажмите export

3. Программа спросит вас выбрать папку pascal. После этого появиться сообщение

4. Откройте или перезагрузите pascal
